## About

hey, what's up.

who actually looks at github projects anymore? most people just ask AI for code now.

that sounds a bit depressing if you still have that old-school part of you that likes to open **vim at 2am**, write some code slowly, and enjoy the feeling of talking directly to the computer (not to AI).

don't get me wrong. i use AI every day like everyone else.

but this profile is a bit different.

here you will find different kinds of projects:

* projects from my time at **zone01 oujda** (01-edu curriculum)
* small experiments
* half-finished ideas
* low-level stuff
* things i started just because they sounded interesting

many repos are unfinished. some are messy. some might stay that way forever.

there are things like:

* a bootloader
* networking experiments
* my programming language
* assembly code here and there
* rust projects
* random systems programming ideas

some of these projects don't even exist yet. they're still somewhere in my head.

these are the kinds of things i work on late at night, when i have the freedom to write slowly, think about the details, and build things piece by piece with minimal AI help.

for me, writing code like that is a kind of meditation.

this profile is not meant to impress recruiters or show perfect projects. it's just a place where i explore ideas and enjoy the process of building things.

if you also like writing code for the sake of writing code, feel free to visit from time to time.

good luck navigating the repos.
